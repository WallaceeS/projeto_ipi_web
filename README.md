Este é um projeto acadêmico desenvolvido para as avaliações (P1 e P2) da disciplina de Programação Web. 

## O que tem no projeto:
* Lista de produtos com barra de pesquisa.
* Filtro de categorias (Tênis e Bolas).
* Carrinho de compras funcionando no LocalStorage.
* Simulação de tela de Login e Recuperação de Senha.
* Layout responsivo para rodar no celular.

## Tecnologias Usadas:
* Angular 17+
* Bootstrap 5
* HTML / CSS / TypeScript
